# Ingeniería en Sistemas / Informática

## Descripción

Carrera orientada al diseño, desarrollo, implementación y gestión de sistemas de software complejos. Forma profesionales capaces de abordar problemas tecnológicos a gran escala, integrando software, hardware, redes y procesos organizacionales.

---

## Objetivos

- Diseñar arquitecturas de software robustas
- Gestionar proyectos tecnológicos
- Desarrollar sistemas escalables
- Integrar soluciones tecnológicas en organizaciones

---

## Áreas principales

- Programación (estructurada, orientada a objetos)
- Estructuras de datos y algoritmos
- Bases de datos
- Sistemas operativos
- Redes
- Ingeniería de software
- Arquitectura de sistemas
- Gestión de proyectos

---

## Tecnologías típicas

- Lenguajes: Java, C#, Python, Go
- Backend: Node.js, Spring Boot
- Bases de datos: PostgreSQL, MySQL, MongoDB
- DevOps: Docker, Kubernetes (no siempre en profundidad)

---

## Salidas laborales

- Backend Developer
- Software Engineer
- Systems Architect
- Tech Lead
- DevOps Engineer (con especialización)

---

## Ventajas

- Formación generalista sólida
- Alta salida laboral
- Base fuerte para especialización

---

## Desventajas

- IA y data suelen ser superficiales
- Infraestructura moderna no siempre bien cubierta

---

## Relación con capas

- Capa 3 (Backend): FUERTE
- Capa 2 (Infraestructura): MEDIA
- Capa 5 (IA): BAJA (requiere complemento)

---

## Perfil recomendado

Personas interesadas en:

- construir sistemas
- resolver problemas complejos
- trabajar en backend o arquitectura

---

## Complementos recomendados

- MLOps
- Cloud (AWS/GCP)
- Sistemas distribuidos
