# Ingeniería en Ciencia de Datos

## Descripción

Carrera enfocada en el análisis, procesamiento y explotación de datos para generar valor mediante modelos matemáticos y técnicas de machine learning.

---

## Objetivos

- Analizar grandes volúmenes de datos
- Construir modelos predictivos
- Diseñar pipelines de datos
- Aplicar inteligencia artificial en problemas reales

---

## Áreas principales

- Matemática (álgebra, cálculo)
- Estadística
- Probabilidad
- Machine Learning
- Data Mining
- Big Data
- Visualización de datos

---

## Tecnologías típicas

- Python (principal)
- Librerías: pandas, scikit-learn, TensorFlow, PyTorch
- Big Data: Spark, Hadoop
- Bases de datos: SQL, NoSQL
- Herramientas: Jupyter, MLflow

---

## Salidas laborales

- Data Engineer
- Data Scientist
- Machine Learning Engineer
- BI Analyst

---

## Ventajas

- Alta demanda en mercado
- Fuerte en IA aplicada
- Buen puente hacia machine learning

---

## Desventajas

- Backend débil
- Infraestructura limitada
- Riesgo de perfil "teórico sin producción"

Personas interesadas en:
