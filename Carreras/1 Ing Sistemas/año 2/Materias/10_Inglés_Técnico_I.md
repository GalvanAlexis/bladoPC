# Inglés Técnico I: Lectura de Documentación Técnica

## Introducción
El inglés técnico es fundamental para acceder a documentación oficial, tutoriales y recursos de programación en su idioma original.

---

## 1. Vocabulario Básico de Programación

### 1.1 Términos Comunes
- **Variable**: variable
- **Function**: función
- **Class**: clase
- **Method**: método
- **Loop**: bucle
- **Array/List**: arreglo/lista
- **Conditional**: condicional
- **Exception**: excepción
- **Debug**: depurar
- **Framework**: marco de trabajo
- **Library**: biblioteca

### 1.2 Verbos Técnicos
- **To implement**: implementar
- **To compile**: compilar
- **To execute**: ejecutar
- **To declare**: declarar
- **To initialize**: inicializar
- **To instantiate**: instanciar
- **To inherit**: heredar
- **To override**: sobreescribir

---

## 2. Lectura de Documentación

### 2.1 Estructura Típica
- **Getting Started**: Primeros pasos
- **Installation**: Instalación
- **Quick Start**: Inicio rápido
- **API Reference**: Referencia de API
- **Examples**: Ejemplos
- **Tutorials**: Tutoriales

### 2.2 Estrategias de Lectura
1. **Skimming**: Lectura rápida para entender la idea general
2. **Scanning**: Búsqueda de información específica
3. **Intensive reading**: Lectura detallada de secciones críticas

---

## 3. Interpretación de Sintaxis

### 3.1 Ejemplo: Documentación de Python
```
def function_name(param1: type, param2: type) -> return_type:
    """
    Brief description.
    
    Args:
        param1 (type): Description.
        param2 (type): Description.
    
    Returns:
        return_type: Description.
    """
```

### 3.2 Ejemplo: Documentación de Java
```java
/**
 * Brief description.
 * 
 * @param param1 Description
 * @param param2 Description
 * @return Description
 * @throws ExceptionType Description
 */
```

## 4. Recursos Recomendados

### 4.1 Documentación Oficial
- **Python**: docs.python.org
- **Java**: docs.oracle.com
- **C++**: cppreference.com
- **Git**: git-scm.com/doc

### 4.2 Práctica
- Leer documentación de librerías utilizadas
- Seguir tutoriales en inglés
- Leer artículos técnicos en Medium, Dev.to
- Ver videos técnicos con subtítulos en inglés

---

## 5. Ejercicios Prácticos

### 5.1 Lectura Comprensiva
1. Leer la documentación oficial de Python sobre funciones
2. Identificar palabras clave y su significado
3. Resumir en español los puntos principales

### 5.2 Traducción Técnica
- Traducir comentarios de código en inglés
- Documentar funciones propias en inglés
- Escribir README.md en inglés para proyectos

---

## 6. Consejos
- Usar herramientas como DeepL o Google Translate para apoyo
- No traducir palabras técnicas (dejar en inglés)
- Practicar con código real, no solo teoría
- Crear un glosario personal de términos
