# Lenguaje: C++

## Introducción
C++ es un lenguaje de programación de propósito general, desarrollado como extensión de C. Es conocido por su alto rendimiento y control de memoria manual.

---

## 1. Características Principales

### 1.1 Compilado y Eficiente
C++ se compila directamente a código máquina, ofreciendo rendimiento crítico.

### 1.2 Programación Multiparadigma
Soporta: programación procedimental, orientada a objetos y genérica.

### 1.3 Control de Memoria
Gestión manual de memoria mediante `new`/`delete` y punteros.

## 3. Sintaxis Básica

### 3.1 Estructura de un Programa
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hola, Mundo!" << endl;
    return 0;
}
```

### 3.2 Variables y Tipos
```cpp
int edad = 25;              // Entero
double precio = 99.99;      // Decimal
bool activo = true;         // Booleano
string nombre = "Blado";    // Cadena (requiere #include <string>)
char letra = 'A';          // Carácter
```

### 3.3 Estructuras de Control
```cpp
// Condicional
if (edad >= 18) {
    cout << "Mayor de edad" << endl;
}

// Bucle
for (int i = 0; i < 5; i++) {
    cout << i << endl;
}
```

## 4. Programación Orientada a Objetos en C++

### 4.1 Clases y Objetos
```cpp
#include <iostream>
#include <string>
using namespace std;

class Persona {
private:
    string nombre;
    int edad;
    
public:
    // Constructor
    Persona(string n, int e) : nombre(n), edad(e) {}
    
    // Método
    void saludar() {
        cout << "Hola, soy " << nombre << endl;
    }
};

// Uso
Persona p("Blado", 25);
p.saludar();
```

### 4.2 Herencia
```cpp
class Estudiante : public Persona {
private:
    string carrera;
    
public:
    Estudiante(string n, int e, string c) : Persona(n, e), carrera(c) {}
};
```

### 4.3 Punteros y Referencias
```cpp
int x = 10;
int* ptr = &x;      // Puntero a x
int& ref = x;       // Referencia a x

cout << *ptr << endl;  // Acceder al valor (10)
cout << ref << endl;    // Acceder al valor (10)
```

## 5. Standard Template Library (STL)

### 5.1 Vectores (std::vector)
```cpp
#include <vector>

vector<int> numeros = {1, 2, 3, 4};
numeros.push_back(5);
int primero = numeros[0];
```

### 5.2 Mapas (std::map)
```cpp
#include <map>

map<string, int> edades;
edades["Blado"] = 25;
int edad = edades["Blado"];
```

## 6. Gestión de Memoria

### 6.1 New y Delete
```cpp
int* ptr = new int(42);   // Asignar memoria
delete ptr;               // Liberar memoria

int* arr = new int[10];   // Arreglo dinámico
delete[] arr;             // Liberar arreglo
```
